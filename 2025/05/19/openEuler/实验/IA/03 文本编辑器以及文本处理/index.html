Vim 是一款功能强大的编辑器，指令和操作的种类非常多。本实验主要涉及到 Vim 常用指令的使用，以帮助学员熟练使用 Vim 编辑器。本实验是以 Vimtutor 教程为基础，介绍在华为openEuler 平台上操作使用 Vim 编辑器的综合实验。本实验指导书共包含 8 个实验，从使用华为 openEuler 实验环境开始，逐一介绍了 Vim 的基本操作、Vim 文件操作等内容。

  • 实验一为使用本地 openEuler 实验环境；
  • 实验二为 Vim 基本操作；
  • 实验三为 Vim 操作符和动作；
  • 实验四为 Vim 修改操作；
  • 实验五为 Vim 搜索替换操作；
  • 实验六为 Vim 文件操作；
  • 实验七为 Vim 其他操作；
  • 实验八为使用 Vim 帮助文档。

实验目标

  • 熟悉 vi 与 vim 编辑器；
  • 掌握 vim 编辑器的安装；
  • 掌握 vim 编辑器不同模式的切换；
  • 掌握 vim 编辑器的常用操作。

3.1.2 Lesson 2.1 删除命令

Type d$ to delete to the end of the line. Type dw to make the word disappear.`

Many commands that change text are made from an operator and a motion.
The format for a delete command with the d delete operator is as follows:

华为的文档介绍了dw,de,d$的操作

  • dw - 从光标删除到下一个单词的开头，不包括他的第一个字符
  • de - 从光标删除到当前单词的结尾，包括他的最后一个字符
  • d$ - 从光标删除到这一行的最后，包括最后一个字符

3.1.3 操作符和动作

d motion

  • d - is the delete operator.
  • motion - is what the operator will operate on (listed below).
    A short list of motions:
  • w - until the start of the next word, EXCLUDING its first character.
  • e - to the end of the current word, INCLUDING the last character.
  • $ - to the end of the line, INCLUDING the last character.

其中“d”表示删除操
作符，“motion”表示操作的动作。“de”删除光标到下一个词的结尾的内容，“dw”删除光标到下一个词的开始的内容，“d$”删除光标到所在行最后的内容。

3.1.4 在动作前添加数字

  • 输入“2w”使光标向右移动两个单词。
  • 输入“3e”使光标向右移动到第三个单词后。
  • 输入“0”使光标移动回到该行开头位置。

w和e的区别在于w移动到单词的开头，e移动到结尾

3.1.5 在删除命令前添加数字

In the combination of the delete operator and a motion mentioned above you insert a count before the motion to delete more:

输入“d2w”删除“ABC”和“DE”这两个大写单词。

输入“dd”删除第二行
移动光标至第 4 行，输入“2dd”删除第 4、5 两行。

3.1.7 撤销操作

Press u to undo the last commands, U to fix a whole line.

  • u撤回
  • 按“Shift”和“U”键撤销这一行所有操作，返回到编辑前的初始状态。
  • 按住“Ctrl”键后多次按“R”，可以重做刚刚“u”命令的撤销操作，使句子恢复删除多余字母后的状态。

3.3 Vim 搜索替换操作

3.3.1 光标位置和文件状态

光标位置和文件状态
按“Ctrl”和“G”键，显示文件名和位置信息，并记录行号 492

输入“gg”跳转到文件开头。

3.3.2 搜索命令

按“/”键进入搜索框，输入“errroor”后按回车键进行搜索。
按“n”键再次搜索“errroor”，按“Shift”和“N”键反方向再次搜索“errroor”，通过“Ctrl”和“O”命令移动到之前位置，“Ctrl”和“I”命令移动到之后位置。

3.3.3 搜索匹配插入语

Type % to find a matching ),], or }

再次输入“%”返回之前“(”的位置。
移动光标至[等其他括号位置，使用“%”进行搜索匹配的括号

3.3.4 替换命令

Type :s/old/new/g to substitute ‘new’ for ‘old’.

向下移动光标，至能看全 4.4 节的内容，移动光标至—>第一行开头的位置。
输入:s/thee/the后按回车键，将第一个“thee”替换为“the”。
输入:s/thee/the/g后按回车键，将行内全部的“thee”替换为“the”。
%s/word1/word2/gi 将文档所有的word1替换为word2，不区分大小写。
type :#,#s/old/new/g where #,# are the line numbers of the range of lines where the subsitution is to be done.
type :%s/old/new/g to change every occurrence in the whole file.
type :%s/old/new/gc to find every occurrence in the whole file,with a prompt wether to substitute or not.

3.4 Vim文件操作

3.4.1 执行外部命令

Type :! followed by an external command to execute that command.

输入“:!ls”，通过“!”符号执行外部 Shell 命令，如 ls 列出目录命令。按回车键返回教程文档。

3.4.2 更多写文件操作

To save the changes made to the text, type :w FILENAME.
输入“:w TEST”在当前目录新建保存名为“TEST”的文件

再次输入“:!ls”查看目录，已新建名为“TEST”的文件。

3.4.3 选择文本写入文件

To save part of the file, type v motion :w FILENAME
输入“v”进入可视化模式，移动光标至第“5”行后。
输入“:w! TEST”后按回车键，把高亮的内容强制写入之前新建的名为“TEST”的文件。

输入“:!cat TEST”查看 TEST 文件的内容，再按回车键返回教程文档。

3.4.4 取回文件内容

To insert the contents of a file, type :r FILENAME

Use the y operator to copy text and p to paste it

3.5.5 设置选项

Set an option so a search or substitute ignores case

输入“/ignore”后按回车键，按“n”键进行多次搜索，按“Shift”和“n”进行反向搜索。

向下移动光标，至能看全 6.5 节的内容。

Typing “:set xxx” sets the option “xxx”. Some options are:
847 ‘ic’ ‘ignorecase’ ignore upper/lower case when searching
848 ‘is’ ‘incsearch’ show partial matches for a search phrase
849 ‘hls’ ‘hlsearch’ highlight all matching phrases
850 You can either use the long or the short option name.

输入:set ic后按回车键，按n键进行多次搜索，按Shift和n进行反向搜索。此时的搜索结果已经忽略了大小写，Ignore和IGNORE都可以被搜索到。
输入:set hls is后按回车键，搜索的内容有高亮，按n键进行多次搜索，按Shift
和n进行反向搜索。
输入:set noic可以取消忽略大小写，输入:nohlsearch可以取消高亮搜索。

向下移动光标，至能看全 7.1 节的内容。
退出文档 输入vim指令打开
输入“:help”并按回车键。
进入 help.txt 帮助文件主页面。
将光标移动到第 10 行 57 列绿色的“bars”位置。

按下“CTRL”和“]”键，跳转到 bars 标签位置。

按下“CTRL”和“T”或者“CTRL”和“O”键，返回到 bars 位置。

输入“:set mouse=a”并按回车键。（？）
鼠标双击“bars”位置，跳转到 bars 标签位置。

  • 输入“g”后点击鼠标右键，返回到 bars 位置。
  • 移动光标至第 20 行 60 列位置，输入“:help x”并按回车键，移动光标查看删除文本命令x 的帮助文档，查看完成后可以使用“CTRL”和“O”键返回之前浏览的位置
  • 可以通过输入“:help word”后按“CTRL”和“D”来查找与“word”相匹配的内容。
  • 继续输入“-count”后按回车，查看“word-count”相关内容。
  • 按回车键返回，移动光标至第 39 行，使用“CTRL”和“]”，“CTRL”和“O”的方式查看“quickref”快速参考以及后面的“USER MANUAL”用户手册。
  • 输入“:q”后按回车键退出帮助文档，再次输入“:q”后按回车键退出 Vim。
