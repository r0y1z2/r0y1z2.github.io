1.1 Apache安装及测试

1.1.1 配置步骤

  1. 安装httpd dnf install -y httpd

  2. 启动httpd并测试 httpd -v

  3. 使用命令httpd -t检查配置文件是否正确

    报错信息：AH00558: httpd: Could not reliably determine the server’s fully qualified domain name, using fe80::f21:38fa:dc6d:d4e9%ens33. Set the ‘ServerName’ directive globally to suppress this message
    Syntax OK

  4. 修改配置文件（/etc/httpd/conf/httpd.conf）中第 97 行关于 ServerName 的配置，消除报错信息，具体如下
    vim /etc/httpd/conf/httpd.conf
    /ServerName 搜索关键字 N搜索下一个

    :wq保存并退出
    http -t 重新检查配置文件是否正确

    使用命令 systemctl enable httpd --now(–now Start or stop unit after enabling or disabling it) 启动 httpd 服务，并将其设置为开机启动，然后使用命令systemctl status httpd检查httpd服务状态
